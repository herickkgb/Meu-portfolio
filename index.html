Projetos Recentes

TaskApp

Como estudante, desenvolvi um projeto durante minha jornada de aprendizado. A proposta desse projeto foi criar o aplicativo de gerenciamento de tarefas TaskApp, que é totalmente digital e estará disponível gratuitamente na Play Store. O projeto envolveu a integração do Firebase para o banco de dados em tempo real, bem como o uso do navigator e adaptadores para listas. O TaskApp está plenamente funcional, oferecendo aos usuários uma ampla gama de funcionalidades, como adição de tarefas, notificações, categorização de tarefas, agendamento, acompanhamento do progresso e a opção de definir lembretes. Você também pode acessar o código-fonte do aplicativo no GitHub. Clique na imagem abaixo para obter mais detalhes sobre o TaskApp.

Clone Netflix

A proposta deste projeto é a criação de um aplicativo que funcione como um clone do famoso serviço de streaming Netflix. Os recursos incluirão a capacidade de criar uma conta de usuário, editar perfil, reproduzir vídeos em tempo real, salvar conteúdos favoritos, receber recomendações personalizadas. Durante minha jornada de estudos, desenvolvi este projeto com o objetivo de aprimorar minhas habilidades em desenvolvimento móvel, especialmente no contexto de aplicativos de entretenimento.
